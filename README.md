# CMSC 207 Mini Project – Personal Web Profile

This project is a personal web profile website created as part of the CMSC 207 course requirement. The goal of the project is to build a simple single-page website using HTML, CSS, and JavaScript while demonstrating basic web development concepts such as layout design, navigation, styling, and interactivity.

The website introduces basic information about me, the things I enjoy, and fun facts about my cats.

---

## Live Demo

You can view the website here:
https://nenlimbauan.github.io/cmsc207-personal-web-profile/

---


## Technologies Used

- **HTML** – Used to structure the content of the webpage
- **CSS** – Used for styling, layout, and responsive design
- **JavaScript** – Used to add interactivity, including navigation highlighting and the button that reveals my cat story

---

## Features

- Single-page personal website layout
- Navigation menu with anchor links
- Responsive layout for desktop and mobile devices
- Interactive button that reveals a fun fact about me
- Cat photo gallery with hover/tap descriptions
- Clean and simple visual design
